CSS Transorm allow modification of an element shape and position. It can perform operation like rotate, scale, skew and translate. Does not effect the layout of surrounding elements. Used to create visual effects.
    Rotate
  • Rotate an element around a fixed point.
  • Defined using the rotate() function withing the transform property.
  • Default rotation point is the element center.
  • rotatex('angel') and rotatey('angel') doesn't decrease the size of element, it just hide the height, just like a pen: if we will move the pen on x and y axes it will hide from user, but size will be same
    • Rotate (45deg)
    Scale
  • Scale will increase and decrease an element's size.
  • scale(factor) => factor: (2 times)twice, 4 times, 5 times etc...
    • scale(factor)
    Translate
  • Scale will shift an element's from it's position .
  • translate(value1 px=) OR translate(value1 px, value2 px) OR translate(-value1 px, -value2 px)
  • translate can take both positive and negative values.
    • translate(value in px)
