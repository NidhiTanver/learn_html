• Element Selector
    1. If i am selecting any element with their HTML tag like h1, p. div etc h1 { color: red; } then it's the element selector.
  • ID Selector
    1. If i am selecting any element with their ID like #header, #main, #footer etc #header { color: red; } then it's the ID selector.
  • Class Selector
    1. If i am selecting any element with their class like .header, .main, .footer etc .header { color: red; } then it's the class selector.
  • Universal Selector
    1. If i am selecting all the elements with * then it's the universal selector. * { color: red; }
  • Descendant Selector

    • If i am selecting any element with their parent like div, p,h4 etc
